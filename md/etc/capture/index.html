デスクトップのスクリーンショット

Windows のデスクトップ上で取得したアクティブウィンドウのスクリーンショットは、クリップボードからそのまま GIMP に貼り付けて利用することができます。手順は下記の通りです。

  1. Windows のデスクトップ上で取得したいウィンドウを選択し、アクティブにする
  2. キー Alt + Print Screen を押下
  3. GIMP の画像ウィンドウ上で、キー Ctrl + V を押下

Warning

取得したはずのウィンドウが一部しか表示されない場合、キャンバスサイズが小さすぎる可能性があります。 メニュー 画像 > キャンバスをレイヤーに合わせる を選択してみてください。

Tip

スクリーンショットが取得した状態であれば、メニューバー ファイル > 画像の生成 > クリップボードから でも GIMP に取り込むことができます。

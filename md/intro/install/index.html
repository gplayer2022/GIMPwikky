ダウンロードとインストール

ダウンロード

窓の杜、または公式サイト（英語）からダウンロードできます。 特に理由がなければ、窓の杜からのダウンロードで問題ありません。

Tip

ポータブル版は PortableApps.com からダウンロードできます。

インストール

インストール用のセットアップファイルを実行してください。

すべてのユーザー用にインストールするか、現在のユーザー用にインストールするかを選択できます。

インストール中に使用する言語を選択します。日本語も選択可能です。

特にカスタマイズが必要なければ、そのままボタン インストール を押下して、インストールを開始します。

数分でインストールが完了します。

Tip

ポータブル版はダウンロードしたファイルをダブルクリックすると、展開先を訊かれます。 好きな場所に展開してください。インストール版とは異なり、後からフォルダごと移動することもできます。

インポートと開く

GIMP が扱うファイルは大きく分けて 2 種類あります。 ひとつは拡張子 .xcf を持つ編集ファイルです。編集ファイルには、編集中の状態がそのまま保存されています。 もうひとつは一般的な画像ファイルで、拡張子は .jpg や .png です。

編集ファイルを読み込むことを開くと言い、それ以外のファイル、画像ファイルなどを読み込むことをインポートと言います。 いずれの操作も下記の手順で実行できます。

  1. メニューバー ファイル > 開く/インポート を選択
  2. ウィンドウ 画像ファイルを開く でファイルを選択
  3. ウィンドウ 画像ファイルを開く で、ボタン 開く を押下

Tip

編集ファイルや画像ファイルを、エクスプローラーから画像ウィンドウに、直接ドラッグ＆ドロップすることもできます。

Tip

一度読み込んだことのあるファイルは、ドック上部でタブ 最近開いたファイル を選択し、一覧からファイル名をダブルクリックすることでも開くことができます。

Tip

GIMP は、拡張子 .psd を持つ Photoshop 形式のファイルをインポートすることもできます。
